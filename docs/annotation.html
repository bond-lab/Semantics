Phase 1: Annotation Task

The format is:

Group ID: Sentence IDs FROM–TO

ID Name Corpus

For example:

Group 2221B: Sentence IDs 0–222
s1 Herlock SHOLMES CorpusA
s111 John Hamish WATSON CorpusB

  • The id is for internal use, please use it to log in.
  • Clicking on the corpus jumps you to the start sentence (with 4 sentences before and after)
  • See Project 1 for the instructions.
  • You will have to log in each time you change to a new browser
  • Don't tag past the end sentence (there will not be a warning, so watch the sentence number)
  • Sentences vary in length, so some have more concepts, some less, but you all have a little over 200 concepts to tag.
